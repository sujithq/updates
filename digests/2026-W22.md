## Executive summary

This week’s news was dominated by the continued shift from AI assistants to governed, production-ready agents across Microsoft Foundry, Visual Studio, GitHub Copilot, and Azure application platforms. GitHub shipped a dense set of Copilot and platform updates, including new model options, task-based auto model selection, semantic issue search, and improvements to code review workflows, while also disclosing an investigation into unauthorized repository access.
