# Weekly Digest: July 06 – July 13, 2026

## Executive summary

This week’s announcements centered on AI agents becoming more operational, governable, and cost-aware across Microsoft 365 Copilot, GitHub Copilot, and Microsoft Foundry. GitHub’s updates stood out for developer productivity, with new Copilot models, repository understanding features, broader code quality and security capabilities, and a generally available pull requests dashboard. On Azure, the most important themes were agent observability, Logic Apps and platform operations improvements, security hardening, and a steady stream of service GA and preview updates across Event Hubs, Blob Storage SFTP, Chaos Studio, and Log Analytics. For architects, the clearest pattern is convergence: AI features are moving from experimentation into platform-integrated workflows with stronger governance, telemetry, and enterprise deployment controls.

## Copilot & Agents

Microsoft’s Copilot and agent updates focused on model freshness and broader enterprise use of agents in day-to-day work. The messaging continues to shift from standalone AI features toward collaborative, embedded agent experiences.

- Microsoft 365 Copilot added OpenAI’s GPT-5.6.
- Agent 365 published its June 2026 roundup of new capabilities.
- “(Co)work in Progress” highlighted evolving human-agent collaboration patterns in Microsoft 365.

## GitHub

GitHub’s news was heavily focused on Copilot usability, platform governance, and code security, with several meaningful improvements for enterprise teams. The most notable updates improve repository understanding, pull request workflows, model choice, code quality rollout, and telemetry administration.

- CodeQL 2.26.0 added Kotlin 2.4.0 support and AI prompt injection detection.
- OpenAI GPT-5.6 Sol, Terra, and Luna became available in GitHub Copilot.
- The new pull requests dashboard reached general availability.
- Copilot can now provide a repository overview.
- GitHub introduced organization-level targeting for GitHub Code Quality.
- Innersource security advisories are now generally available.
- Enterprise-managed OpenTelemetry export for VS Code and CLI was added.
- GitHub Mobile improved filters and sorting for Copilot sessions.
- Secret scanning detector types now have clearer names.
- GitHub also published platform engineering and reliability updates, including durable repository ownership and the June 2026 availability report.

## Data & AI

Azure’s Data & AI announcements emphasized production readiness for agents and models: pricing, observability, evaluation, ROI, and enterprise data grounding. The strongest signal this week was that Foundry is being positioned as an end-to-end platform for deploying, measuring, and operating agents.

- Microsoft Foundry announced a model deployment pricing update.
- Frontier Tuning was introduced as a shift from classic fine-tuning.

- Support for 5x churn in Azure Site Recovery is now generally available.
