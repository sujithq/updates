# Weekly Digest: Azure, GitHub, and Developer Ecosystem  
**Period: April 27 – May 04, 2026**

## Executive summary

This week’s announcements centered on three major themes: agent platforms maturing quickly, developer tooling adapting to AI-native workflows, and platform teams tightening governance, security, and cost control around those changes. Microsoft pushed forward on Foundry Agent Service, Azure SRE Agent, and Microsoft 365 Copilot capabilities, while GitHub signaled a major commercial and operational shift with usage-based Copilot billing and updated model availability. On the platform side, retirements and migrations stood out—especially Prompt flow and Azure Data Studio—alongside new guidance for moving to VS Code, modern IaC patterns, and cloud-scale observability. Kubernetes v1.36, storage and networking GA updates, and multiple security releases reinforced that reliability and control remain top priorities as AI features expand.

Copilot and agent announcements this week focused on broader capability expansion, operational readiness, and more specialized workflows across Microsoft 365 and security scenarios. The clearest pattern is that agent experiences are moving from novelty to governed, resilient, production use.
