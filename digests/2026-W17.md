# Weekly Digest: Azure, GitHub, and Developer Ecosystem  
**April 13–20, 2026**

## Executive summary

This week’s announcements were dominated by AI platform and agent development momentum across Microsoft and GitHub, with Claude Opus 4.7 showing up across Microsoft 365 Copilot, GitHub, Azure Databricks, and Microsoft Foundry. GitHub focused on practical developer productivity and security improvements, including Copilot CLI enhancements, expanded code scanning and Dependabot capabilities, and better policy and status transparency. On Azure, the biggest themes were operationalizing AI agents on App Service, AKS and infrastructure updates, and a steady drumbeat of platform lifecycle notices developers need to plan around. Across tools, VS Code, Visual Studio, .NET, and Azure DevOps all shipped updates aimed at tightening inner-loop development, debugging, security scanning, and agent-centric workflows.

## Copilot & Agents

Copilot and agent news was light in volume but meaningful in direction: Microsoft is expanding model choice in Microsoft 365 Copilot and continuing to improve document-centric workflows, while also signaling more community engagement around Agent 365.

- Anthropic Claude Opus 4.7 is now available in Microsoft 365 Copilot.
- Agent 365 scheduled a live AMA for deeper product discussion.
- Copilot in Word added new capabilities for document workflows.

## GitHub

GitHub’s week centered on AI-assisted development, security posture, and governance. The standout items were Copilot CLI improvements, broader support for cloud and agent workflows, and multiple security enhancements spanning OIDC, code scanning, Dependabot, and CodeQL.

- GitHub Copilot CLI now supports automatic model selection.
- GitHub added GitHub CLI support for managing agent skills.
- Claude Opus 4.7 reached general availability on GitHub.
- CodeQL 2.25.2 added Kotlin 2.3.20 support and other updates.
- OIDC support expanded to Dependabot and code scanning.
- Organizations can now use private registries for Dependabot and code scanning at org level.
- Code scanning alerts can now be linked directly to GitHub Issues.
- Deployment context now appears in repository properties and alerts.
- GitHub introduced a rule insights dashboard and unified filter bar.
- GitHub published a developer policy update covering intermediary liability, copyright, and transparency.

## Azure Updates

Azure Updates mixed key launches with several important retirement notices that platform teams should track now. The biggest operationally relevant items were AKS backup GA, new AKS monitoring and networking previews, storage enhancements, and multiple deprecation timelines across Functions, AKS, and Managed Grafana.

- Anthropic Claude Opus 4.7 is now generally available on Azure Databricks.
- Azure Functions runtime v3 on Linux Consumption will stop running on September 30, 2026.
- Configuring AKS backup with a single Azure CLI command is now generally available.
- AKS support for Ubuntu 22.04 is being retired.
- StandardV2 NAT Gateway as an outbound type for AKS entered public preview.
- Monitoring AKS applications with OpenTelemetry and Azure Monitor entered public preview.
- Cross-tenant CMK encryption for Premium SSD v2 and Ultra Disks is now generally available.
- Smart Tier for Azure Blob and Data Lake Storage is now generally available.
- Granular encryption-in-transit controls for SMB and NFS on Azure Files are now generally available.
- Azure Managed Grafana Essential SKU will retire on March 30, 2027.
