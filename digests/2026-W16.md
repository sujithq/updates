# Weekly Digest: Azure, GitHub, and Developer Ecosystem  
**April 06 – April 13, 2026**

## Executive summary

This week’s news centered on the rapid operationalization of agentic AI across Microsoft’s stack: Foundry, App Service, Azure Functions, AKS, and Microsoft 365 all added capabilities for building, grounding, governing, and observing AI agents. GitHub’s updates focused on Copilot controls, usage visibility, and incremental product hardening, while developer tooling emphasized MCP, Azure Functions integration, and agent-oriented workflows in VS Code, Aspire, and the Azure SDK. On the Azure platform side, App Service, Azure Arc, AKS, and OpenShift all saw meaningful improvements for identity, deployment, observability, and infrastructure automation. Security coverage remained significant as Microsoft paired product updates in Sentinel and Defender for Cloud with fresh threat intelligence on phishing, ransomware, DNS hijacking, and Android wallet exposure.

## Copilot & Agents

Microsoft 365 Copilot updates this week were about making Copilot more practical in day-to-day work while improving admin visibility and controls. The emphasis was on real-world usage patterns, better document workflows, and stronger management and analytics.

- Microsoft 365 Copilot app learning series highlighted real-world scenarios for everyday work.
- Copilot in Word introduced new capabilities for document workflows.
- Microsoft detailed the latest enhancements for Copilot security, management, and analytics.

## GitHub

GitHub’s announcements focused on tightening Copilot plans and limits, broadening usage reporting, and polishing core collaboration workflows. For teams standardizing on Copilot, the most relevant changes were around metrics, CLI support, and security assessment integration.

- GitHub paused new GitHub Copilot Pro trials.
- Copilot usage metrics now aggregate Copilot cloud agent active user counts.

- Actions workflows are now limited to 50 reruns.
- GitHub also added smaller UX improvements across notifications, comments, and pull request list views.
