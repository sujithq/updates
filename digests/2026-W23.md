# Weekly Digest: Microsoft Azure, GitHub, and Developer Ecosystem

## Executive summary

This week’s announcements centered on three big themes: tighter AI platform integration, stronger governance for Copilot and GitHub usage, and steady Azure infrastructure maturation. Microsoft expanded Claude Opus 4.8 availability across both Microsoft 365 Copilot and Microsoft Foundry, while a broad set of Foundry posts focused on production readiness, observability, migration, and agent deployment patterns. GitHub’s updates emphasized enterprise controls and measurement, with new Copilot usage cohorts, model targeting rules, memory controls, and budget limits for Advanced Security. On the Azure side, notable items included GA for Intel TDX confidential VMs, Virtual Network Manager integration with Virtual WAN in preview, and service mesh integration for Application Gateway for Containers.
