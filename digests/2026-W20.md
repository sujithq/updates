# Weekly Digest: May 04 – May 11, 2026

## Executive summary

This week’s announcements centered on AI platform expansion, agent governance, and infrastructure modernization across Microsoft and GitHub. Microsoft pushed new model and multimodal capabilities into Foundry, advanced Azure SQL and Cosmos DB for AI workloads, and continued investing in agent frameworks, MCP servers, and enterprise-grade deployment patterns. GitHub’s updates were notably operational, with model deprecations, Copilot metrics improvements, agent workflow guidance, and policy controls for repositories and migrations. On the platform side, Azure infrastructure and update channels brought important VM, storage, migration, and Kubernetes changes that matter for scale, resiliency, and lifecycle planning.

## Copilot & Agents

Microsoft 365 Copilot updates focused on faster foundation models, broader enterprise data access, and ongoing agent adoption across work scenarios. The category signals continued movement from chat experiences toward connected, task-oriented copilots and agents.

- GPT-5.5 Instant is now available in Microsoft 365 Copilot.
- Federated Copilot connectors bring real-time enterprise data into Microsoft 365 Copilot.
- April 2026 Microsoft 365 Copilot updates summarize recent product additions and changes.
- Agent 365 is hosting a live AMA on May 12 for questions from customers and practitioners.
- “(Co)work in Progress” highlights evolving collaboration patterns around Copilot and agents.

## GitHub

GitHub’s week was dominated by model lifecycle changes, enterprise administration improvements, and maturing agentic development workflows. For teams using Copilot and GitHub at scale, the biggest themes were deprecations, governance controls, and better observability.

- GPT-4.1, Claude Sonnet 4, and Grok Code Fast 1 are all headed for deprecation.
- Copilot code review comment types are now exposed in the usage metrics API.
- Copilot cloud agent now supports more flexible secrets and variables.
- Enterprise Live Migrations entered public preview.
- Repository rulesets gained user bypass and branch renaming support.
- CodeQL 2.25.3 adds Swift 6.3 support.
- GitHub Copilot CLI’s Rubber Duck now supports more models.
- GitHub published guidance for reviewing agent pull requests.
- GitHub shared work on improving token efficiency in Agentic Workflows.
- GitHub also covered validation techniques for agentic behavior when outputs are nondeterministic.

## Data & AI

Data and AI announcements were especially strong, with new models in Foundry, multimodal and realtime AI enhancements, and better platform support for enterprise-scale retrieval and automation. Azure SQL, Cosmos DB, and Databricks all saw updates aimed at production AI systems rather than experimentation alone.

- Azure SQL general availability now includes Generate Embeddings Function and External Model Object support.
- Microsoft Foundry added DeepSeek V4 Flash and V4 Pro.
- Foundry also added IBM Granite 4.1, NVIDIA Nemotron Nano Omni, and Qwen3.6-35B-A3B.
- OpenAI’s newest chat model is now available in Microsoft Foundry.
- GPT-image-2 is now available in Microsoft Foundry.
- Azure AI Foundry announced a new chapter for realtime AI with reasoning, translation, and real-time transcription.
- Azure Cosmos DB Shell entered public preview as an open-source, AI-assisted database automation tool.
- Cosmos DB highlighted enterprise-scale generative AI patterns with Super Insight by AVASOFT.
- Azure Databricks guidance emphasized disaster recovery, secure medallion architectures, and workspace visibility.
- New architectural guidance covered LLM-powered backfill agents and Tableau-to-Power BI migration patterns.

## Developer Tools

Developer tooling news emphasized AI-assisted engineering workflows, modern app orchestration, and practical patterns for quality and agent development. Visual Studio Code, .NET, Aspire, azd, and community engineering content all reinforced the move toward agent-aware developer platforms.

- Visual Studio Code 1.120 shipped this week, alongside the prior 1.119 release notes.
- Aspire 13.3 introduced the latest updates for distributed app development.
- Copilot Studio gets faster with .NET 10 on WebAssembly.
- .NET published guidance on durable workflows in the Microsoft Agent Framework.
- Microsoft Agent Framework – Building Blocks for AI Part 3 continued the framework series.
- A new azd template from Curity and Microsoft focuses on least-privilege AI agents.
- Azure Dev Community showed how to turn GitHub Copilot into a best-practices coach with Copilot Spaces.
- Enterprise quality engineering guidance covered using GitHub Copilot to move from test cases to trust.
- Practical posts explored spec-driven development and natural-language SQL query generation.
- Cosmos DB Conf 2026 recap highlighted production lessons for developers and architects.

## Azure Platform

Azure platform announcements focused on integration modernization, Kubernetes operations, storage access, and workload performance. The common thread was simplifying platform operations while expanding support for hybrid and integration-heavy environments.

- cert-manager for Azure Arc-enabled Kubernetes entered public preview.
- Microsoft Host Integration Server 2028 was announced for modern connectivity to IBM mainframes and midrange systems.
- Logic Apps guidance emphasized consolidating integration workloads on Logic Apps Standard.
- Azure Blob Storage updated host key requirements for SFTP usage.
- New guidance covered performance tuning and scaling optimization for large Azure workloads.
- The May 2026 Logic Apps newsletter summarized current platform developments.

## Azure Infrastructure

Infrastructure news was one of the busiest areas this week, spanning migration, AI platform design, deployment reliability, and VM and storage lifecycle changes. Architects should pay particular attention to new migration previews, VM availability changes, and practical automation patterns across Bicep, Terraform, and GitHub Copilot.

- Azure Compute previewed migration of regional virtual machines to availability zones.
- Azure Compute also previewed migration from Availability Sets to Virtual Machine Scale Sets.
- Azure Dl/D/Esv7-series VMs based on Intel Xeon 6 processors reached general availability.
- Azure Reserved VM Instances for select VM series will no longer be available starting July 1, 2026.
- Prefix-scoped access for User Delegation SAS in Azure Blob Storage is now generally available.
- Azure shared guidance for handling the 800 deployment quota limitation in Bicep and ARM.
- Multiple posts focused on making Terraform and Azure Function deployments more predictable and safer.
- Azure published patterns for secure AI platforms, responsible AI in Foundry, and AI infrastructure validation agents.
- Azure Container Registry can now be used as an upstream source for Artifact Cache.
- HPC guidance covered faster model-weight distribution and resilient networks for AI supercomputers.

## Containers & Kubernetes

Kubernetes coverage this week was dominated by upstream v1.36 feature maturity, especially around storage, validation, policy, and scalability. For AKS and platform teams, these changes are most relevant for cluster governance and large-scale operations.

- AKS published its 2026-04-28 release notes.
- Kubernetes v1.36 moved Volume Group Snapshots to GA.
- Kubernetes v1.36 advanced Dynamic Resource Allocation with more drivers and new features.
- Server-Side Sharded List and Watch arrived in Kubernetes v1.36.
- Declarative Validation graduated to GA in Kubernetes v1.36.
- Kubernetes v1.36 introduced admission policies that cannot be deleted.

## Security

Security announcements combined active threat intelligence with platform feature updates for passwordless identity, AI security, and cloud defense integration. The standout message for developers was that agentic systems and prompt-driven tooling are now part of the attack surface.

- Microsoft warned that RCE vulnerabilities in AI agent frameworks can arise when prompts become shells.
- Dirty Frag on Linux was highlighted as expanding post-compromise risk.
- A multi-stage “code of conduct” phishing campaign was shown leading to AiTM token compromise.
- A ClickFix campaign used fake macOS utilities to deliver infostealers.
- World Passkey Day coverage emphasized continued progress toward passwordless authentication.
- Microsoft Sentinel announced RSAC 2026 news and its April 2026 feature roundup.
- Sentinel data integration now includes Azure Blob Storage support for CCF connectors.
- Defender for Cloud highlighted combined protection using Azure WAF, Defender for Storage, and Defender for Azure SQL Databases.
- Microsoft also shared its placement as an overall leader in KuppingerCole’s 2026 emerging AI SOC report.
- The Microsoft Defender for Cloud customer newsletter summarized current product updates.

## Governance & Operations

This week’s governance and operations updates focused on practical control-plane automation and migration execution. Both announcements are directly useful for teams standardizing operations around Azure resource management and PaaS cutovers.

- Azure introduced the Azure Resource Manager MCP Server.
- A step-by-step guide detailed near zero-downtime cutover strategies for Azure PaaS migrations.

## Azure Updates

Azure Updates included several items with immediate planning impact across compute, storage, backup, Cosmos DB, networking, and service retirement. The biggest watchpoints were new previews for backup and containers, plus retirement notices affecting Document Intelligence and reserved VM instances.

- Azure Dl/D/E v7 Virtual Machines are now generally available.
- Azure Functions durable task scheduler Consumption SKU is now generally available.
- Bulk Restore for Azure Virtual Machines using Azure Backup entered public preview.
- Application Gateway for Containers managed add-on with AKS Automatic entered public preview.
- Azure Cosmos DB Shell entered public preview.
- Azure Cosmos DB spherical quantization for improved vector search entered public preview.
- Azure SQL early-May 2026 updates entered public preview.
- Azure NetApp Files can now enable backup by default in public preview.
- Azure Document Intelligence v3.0 API is scheduled for retirement on March 30, 2029.
- Azure Reserved Virtual Machines Instances for select VM series were announced for retirement.

## Ecosystem

The broader ecosystem mixed architecture guidance, open source project progress, resilience design, and a few notable community releases. The strongest signal here was continued work on reference architectures for resilient, governed AI and data platforms on Azure.

- Azure Architecture published guidance for governing agent sprawl with a multi-region AI agent landing zone.
- Azure Architecture also shared a resilience-focused decision guide.
- Azure Architecture covered DNS forwarding for Azure NetApp Files.
- Inspektor Gadget completed its first independent security audit.
- Multitude detailed a resilient banking platform on PostgreSQL and MySQL on Azure.
- Terraform released v1.15.2.
- Community content showed how to send email from Claude Code with zero dependencies.
