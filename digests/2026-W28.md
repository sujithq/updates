# Weekly Digest: Azure, GitHub, and Developer News  
**June 29 – July 06, 2026**

## Executive summary

This week centered on AI agents becoming more operational: Microsoft 365 Copilot added Claude Sonnet 5, sales workflows moved further into general availability, and Azure AI Foundry continued to sharpen its agent architecture story with control planes, A2A communication, and broader model availability. GitHub’s updates were equally consequential, with changes to Copilot telemetry, enterprise AI controls, agent session streaming, secret scanning, and the planned retirement of GitHub Models. Across the platform stack, developers saw meaningful updates in VS Code, Visual Studio, .NET lifecycle guidance, Azure SDKs, and WSL containers, while Azure infrastructure and security teams got notable advances in Blob Storage integrity protections, multicloud Defender coverage, Sentinel capabilities, and Azure landing zone evolution. Overall, the week’s signal was clear: AI features are maturing, governance is tightening, and the surrounding developer tooling is being optimized for production use.

## Copilot & Agents

Microsoft 365 Copilot updates emphasized stronger model choice, more embedded business workflows, and continued movement from chat experiences toward structured, agentic work. The biggest themes were Claude Sonnet 5 availability, sales workflow maturity, and plugins that make Copilot actions more operational.
